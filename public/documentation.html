Mooovies! Documentation Page

This web service API gives users access to a plethora of information about various films, genres, directors, and actors. It will allow users to create personalized accounts as well as lists of favorite movies and movies they would like to see!

The table below lists actions that users can implement with this app:

Allow users to update their user info /users/[username]/profile PUT :username n/a n/a A text message indicating a successful update "Profile successfully updated"
Allow users to add a movie to their list of favorites /users/[username]/favorites POST :username [title] of film to be added to favorites "Field of Dreams" A text message indicating a successful addition "[title] added to favorites"
Allow users to remove a movie from their list of favorites users/[username]/favorites/[title] DELETE :username, :title none none A text message indicating a successful removal "[title] removed from favorites"
Allow existing users to de-register /users/[username] DELETE :username none none A text message indicating successful de-registration "De-registration successful"
